3.4.5 Aliases
template<typename T>
class Vector {
public:
using value_type = T;
// ...
};
In fact, every standard-library container provides value_type as the name of its value type (ยง31.3.1).
This allows us to write code that will work for every container that follows this convention. For
example:
template<typename C>
using Element_type = typename C::value_type;
template<typename Container>
void algo(Container& c)
{
Vector<Element_type<Container>> vec; // keep results here
// ...
}
